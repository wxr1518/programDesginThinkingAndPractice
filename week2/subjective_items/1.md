函数 f1(n)：

- set 插入单个元素复杂度为 O(log k)
- 循环插入 n 个元素，总复杂度为 O(log 1 + ... + log n) ≈ O(n log n)
- 所以 f1(n) 时间复杂度为 O(n log n)

函数 f2()：

- 遍历 set 中 n 个元素
- 每个元素访问为 O(1)
- 所以 f2() 时间复杂度为 O(n)
